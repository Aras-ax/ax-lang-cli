You can reboot the device, restore the device to factory settings, upgrade firmware, backup or restore the configuration of the device here.

Reboot: Clicking "Reboot" button can reboot the device. The device will disconnect all connections during reboot.

Reset to factory settings: Clicking "Reset" button restores the device to factory settings. The default IP address is %s after reset.

Upgrade Firmware: Clicking "Firmware Upgrade" button can upgrade this device.

Backup/Restore: Clicking "Backup/Restore" button can backup the current configuration of the device to local computer, or restore the previous configuration using the configuration file you backup before.
