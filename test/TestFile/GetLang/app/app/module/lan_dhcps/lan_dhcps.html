DHCP Server: select this option if this device is required to assign IP addresses to LAN computers.

Start IP Address and End IP Address: the range of IP addresses that the DHCP server can assign.

Subnet Mask: subnet mask of wireless devices assigned by DHCP server.

Gateway: default gateway IP address of clients assigned by the DHCP server of this device.

Primary DNS Server: mandatory. Set it to the DNS server IP address provided by your ISP. If you are uncertain about the IP address, consult your ISP.

Secondary DNS Server: optional. Set it to the second DNS server address (if any) provided by your ISP.

Lease Time: valid time of IP addresses assigned by DHCP server. If the time expired, devices in LAN need to obtain an IP address again.
