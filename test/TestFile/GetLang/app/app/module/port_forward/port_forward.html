Port mapping function allows the device to map WAN ports to LAN server, and forward the service requests transmitted to these ports to the corresponding server in LAN.

LAN IP Address: IP address of the host which establishes server in LAN.

Internal Port: service port of the server in LAN. A single port is supported.

External Port: ports enabled for WAN users by this device.

Protocol: protocol type of the selected applications.

Application: application services established in LAN.
