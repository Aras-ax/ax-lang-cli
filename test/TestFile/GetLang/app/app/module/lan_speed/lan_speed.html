Auto Negotiation: ports of this device and peer device auto-negotiate their speed and duplex mode to achieve the best performance.

If the Ethernet cable used to connect this device to peer device is longer than 100 meters, you can set the speed of this port to 10 Mbps to increase the driving distance.

Full-duplex mode allows data transmission in both directions simultaneously.

Half-duplex mode allows data transmission in both directions, but only in one direction at a time.
