The device is compliant with IEEE 802.1Q VLAN. With VLAN enabled, this device can work with switches supporting QVLAN and establish multiple wireless subnetworks. Wireless clients belonging to different VLANs cannot communicate with each other.

VLAN Settings: whether to enable the VLAN function of this device.

PVID: ID of the default native VLAN of the trunk port. The default ID is 1.

Management VLAN: ID of the management VLAN of this device. The default ID is 1. After changing the management VLAN, you can manage this device only after connecting your computer to the new management VLAN.

WLAN: allow you to set a VLAN ID for the wireless network of this device.

WLAN: allow you to set a VLAN ID for the wireless network of this device.

LAN: allow you to set a VLAN ID for the LAN port (wired network) of this device.
